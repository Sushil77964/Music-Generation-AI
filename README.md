# 🎵 Music Generation AI

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://music-generation-ai-sushil.streamlit.app/)

An AI-powered Music Generation application built using **Meta MusicGen**, **Hugging Face Transformers**, **PyTorch**, and **Streamlit**.

Generate high-quality music from natural language prompts such as:

- 🎹 A relaxing piano melody
- 🎻 Epic orchestral soundtrack
- 🎸 Soft acoustic guitar music
- 🥁 Happy background music
- 🎼 Meditation and Lo-Fi music

---

# 🌐 Live Demo

🚀 **Try the application online**

```bash

cd Music-Generation-AI

pip install -r requirements.txt
```

